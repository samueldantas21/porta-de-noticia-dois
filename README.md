
Este projeto consiste na criação de um portal de notícias simples, desenvolvido utilizando apenas HTML e CSS. O objetivo é praticar a construção de páginas estáticas, organizando o conteúdo de forma clara e estruturada, além de aplicar conceitos fundamentais de estilização e layout.

O portal apresenta uma página inicial com manchetes, categorias de notícias e seções organizadas, permitindo ao usuário visualizar facilmente os principais conteúdos. Cada notícia é exibida com título, imagem e breve descrição. Não há uso de JavaScript ou backend, mantendo o foco na estrutura semântica e no design responsivo básico criado com CSS.
